# Polar Design README
The project is under production and the high volume of the project does not allow the project to be mirrored on Github. This problem will be soon resolved and the project will be available to be forked.

## Overview

This project is a Unity-based game designed to provide an engaging and interactive experience. This README outlines how to set up, run, and contribute to the project.

## Features

* Core gameplay mechanics implemented in Unity
* Modular scene structure
* Cross-platform build configuration
* Comprehensive input system

## Project Structure

```
Assets/
  Scripts/
  Scenes/
  Prefabs/
  Materials/
  Audio/
  UI/
ProjectSettings/
Packages/
```

## Requirements

* **Unity Version:** 2021.3 LTS or later
* **Platforms Supported:** Windows, macOS, WebGL, Android, iOS

## Getting Started

1. Clone or download the repository.
2. Open Unity Hub and add the project folder.
3. Launch the project with the required Unity version.
4. Open the main scene located in `Assets/Scenes/Main.unity`.

## How to Play

* **Movement:** WASD / Arrow Keys
* **Action Key:** Space
* **Pause:** Esc

## Build Instructions

1. Go to **File > Build Settings**.
2. Select the target platform.
3. Ensure scenes are included in the build.
4. Click **Build**.

## Contributing

1. Fork the repository.
2. Create a new branch for your feature or fix.
3. Commit your changes with clear messages.
4. Submit a pull request.
